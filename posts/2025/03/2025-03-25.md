Modern CSS has evolved dramatically, providing powerful layout tools that make complex designs achievable with clean, maintainable code. Let's explore the essential techniques every developer should know.

## CSS Grid

CSS Grid is the most powerful layout system available in CSS. It's a two-dimensional system, meaning it can handle both columns and rows.

### Basic Grid Setup

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 1rem;
}

.item {
  background: #f0f0f0;
  padding: 1rem;
}
```

### Responsive Grid

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1.5rem;
}
```

### Named Grid Lines

```css
.layout {
  display: grid;
  grid-template-columns: 
    [sidebar-start] 200px 
    [sidebar-end main-start] 1fr 
    [main-end];
  grid-template-rows: 
    [header-start] auto 
    [header-end content-start] 1fr 
    [content-end footer-start] auto 
    [footer-end];
}

.header {
  grid-column: sidebar-start / main-end;
}

.sidebar {
  grid-row: header-end / footer-start;
}

.main {
  grid-area: content;
}
```

## Flexbox

Flexbox excels at one-dimensional layouts and component-level designs.

### Centering Content

```css
.center-container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

### Responsive Card Layout

```css
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.card {
  flex: 1 1 300px;
  min-width: 0;
}
```

### Navigation Bar

```css
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem;
}

.nav-links {
  display: flex;
  gap: 2rem;
  list-style: none;
}
```

## Container Queries

Container queries allow components to respond to their container size, not just the viewport.

```css
.card-container {
  container-type: inline-size;
}

.card {
  background: white;
  padding: 1rem;
}

@container (min-width: 400px) {
  .card {
    display: flex;
    gap: 1rem;
  }
}

@container (min-width: 600px) {
  .card {
    flex-direction: column;
  }
  
  .card-image {
    width: 100%;
  }
}
```

## Subgrid

Subgrid enables nested grids to inherit their parent's grid tracks.

```css
.grid-parent {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 1rem;
}

.grid-item {
  display: grid;
  grid-template-columns: subgrid;
  grid-column: span 2;
}
```

## Logical Properties

Logical properties make layouts work seamlessly in different writing modes.

```css
/* Old approach */
.element {
  margin-left: 1rem;
  padding-right: 1rem;
  border-top: 2px solid black;
}

/* Modern approach */
.element {
  margin-inline-start: 1rem;
  padding-inline-end: 1rem;
  border-block-start: 2px solid black;
}
```

## Modern Techniques

### Aspect Ratio

```css
.image-container {
  aspect-ratio: 16 / 9;
  overflow: hidden;
}

.image-container img {
  width: 100%;
  height: 100%;
  object-fit: cover;
}
```

### Modern Units

```css
.responsive-box {
  width: clamp(300px, 50vw, 600px);
  padding: clamp(1rem, 4vw, 2rem);
}
```

### Grid with Auto Placement

```css
.masonry {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  grid-auto-rows: 10px;
}

.masonry-item {
  grid-row-end: span var(--row-span);
}
```

Mastering these modern CSS layout techniques will give you the tools to create beautiful, responsive, and maintainable layouts for any project.
