# Ask Mode

> Ask Mode lets you explore and understand your codebase without making changes.
>
> Perfect for onboarding, code reviews, or learning how systems work—the agent reads files, searches, and answers questions while keeping your code untouched.

**Docs:** [Ask Mode](https://docs.cursor.com/chat/modes#ask-mode)

## Setup

Switch to Ask Mode by clicking the mode dropdown in the chat panel and selecting "Ask"

### What to Highlight

- Codebase understanding without risk of changes
- Great for onboarding new team members
- Agent reads through files and explains architecture
- No edits are made—purely read-only exploration

---

## Use Cases

- **New to a codebase?** Ask "What does this app do and how is it structured?"
- **Code review prep:** Ask "How does the authentication flow work?"
- **Learning patterns:** Ask "What design patterns are used here?"
